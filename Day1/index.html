Q1 all types of console

1.console.log()

console.log() is used to print object or string etc in console section

2.console.error()

console.error() is used to show the customize error and testing code

3.console.warn()

console.warn() is used to show the customize warning

4.console.time() and console.timeEnd()

console.time() and console.timeEnd() is used to show execution time or how much time taken by that block.In this case we need to pass same string to both function.

5.console.table()

console.table() is used to show the data is table format

6.console.clear()

console.clear() is used to clear console screen

Differnce Between let, var and consta

Var: Var is global veriable.we can access var variables anywhere in code

Let:Let is local veriable. we can use only for perticuler block like for loop etc.

Consta: Consta is shortform of constant. we can't change the constant variable value but it can be mutable
